San Diego Data Handbook

Unidimensional Datasource

A datasource that contains only one dataset. Examples include but not limited to:

  • Shapefile
  • Excel file with only one sheet
  • PDF
  • Word Document

Multidimensional Datasource

A datasource that contains multiple potential datasets. Example include, but not limited to:

  • Excel file with multiple sheets (each sheet is a dataset)
  • GIS server
  • Access Database
